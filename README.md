# dhev.dev

Personal portfolio — web and SSH.

## Web

Next.js app deployed on Cloudflare Pages.

```bash
bun install
bun dev
```

## SSH

Bubbletea TUI served over SSH via Wish.

```bash
cd ssh
go run .
```

```bash
ssh localhost -p 2222
```

## Data

Shared JSON files in `data/` power both frontends.
